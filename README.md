# London GP Directory

A free, data-driven directory of every NHS GP practice in London — searchable by **borough, Primary Care Network (PCN), patient experience scores, and "accepting new patients"** status.

**Live site:** [londongp.directory](https://londongp.directory)

---

## What it does

- **Find a GP near you** — enter any London postcode, see all practices within a chosen radius, sorted by GP Patient Survey scores and distance.
- **Filter by "accepting new patients"** — the single most-searched piece of information for any GP directory. Refreshed weekly.
- **Compare practices side by side** — choose up to 4 practices and compare GPPS scores, list size, workforce ratios, opening hours, and languages spoken.
- **Borough and PCN hub pages** — one rankable page for every London borough and every Primary Care Network.
- **GP Patient Survey results** rendered visually for every practice, with borough and London averages for context.
- **How to register** — step-by-step guides for switching GP, registering without proof of address, and what to do when you cannot get through to your practice.

## Data sources

| Data | Source | Refresh |
|------|--------|---------|
| Practice list, ODS code, address, PCN membership | [NHS Digital ODS](https://digital.nhs.uk/services/organisation-data-service) | Weekly |
| Accepting new patients flag | NHS Digital ODS / NHS website | Weekly |
| GP Patient Survey scores | [Ipsos for NHS England](https://gp-patient.co.uk/) | Annual |
| Workforce (GPs / nurses per 1,000 patients) | [NHS Digital General Practice Workforce](https://digital.nhs.uk/data-and-information/publications/statistical/general-and-personal-medical-services) | Quarterly |
| CQC rating | [Care Quality Commission API](https://api.service.cqc.org.uk/public/v1/) | Monthly |
| Practice list size | NHS Digital Patients Registered at a GP Practice | Monthly |
| PCN structure | [NHS England PCN list](https://www.england.nhs.uk/primary-care/primary-care-networks/) | Quarterly |
| Postcode to coordinate / LSOA | [ONS Postcode Directory](https://www.ons.gov.uk/methodology/geography/geographicalproducts/postcodeproducts) | Quarterly |

All data is open NHS / government data, used under the [Open Government Licence](https://www.nationalarchives.gov.uk/doc/open-government-licence/).

## Tech stack

- **Frontend:** Vanilla HTML + a small React (CDN) island for interactive search on the homepage.
- **Practice pages:** Static HTML generated by `scripts/build_gp_pages.py` — one indexable page per GP practice, no JavaScript required to display the data.
- **Borough / PCN hubs:** Static HTML generated by `scripts/build_gp_borough_hubs.py`.
- **Hosting:** [Vercel](https://vercel.com) — auto-deploys on every commit.
- **Data pipeline:** GitHub Actions — runs weekly for ODS data, monthly for the full rebuild.
- **No database** — every page is a static file for maximum speed, zero ongoing cost, and zero attack surface.

## Refresh pipeline

### Weekly (every Monday 06:00 UTC)
Lightweight refresh: ODS data + accepting-new-patients flag. Catches practice closures, mergers, and the most-searched live attribute. Rebuilds practice pages and updates `lastmod` in the sitemap.

### Monthly (15th of every month, 08:00 UTC)
Full rebuild: ODS + GPPS + workforce + CQC + list sizes. Recomputes every derived metric. Rebuilds all practice, borough, and PCN pages. Updates the sitemap and commits back to GitHub. Vercel auto-redeploys.

Both workflows are public on the [Actions tab](https://github.com/Thierry0303/GP-Directory/actions). If a source is unavailable on refresh day, the previous values are retained and the "data last verified" stamp on the page shows when each metric was last refreshed.

## Running locally

```bash
# Install dependencies
pip install requests pandas numpy pyyaml

# Set your NHS Digital ODS API key (one-time)
export NHS_ODS_API_KEY="your_key_here"

# Fetch fresh data
python3 scripts/refresh_nhs_data.py

# Build practice pages
python3 scripts/build_gp_pages.py

# Build borough + PCN hubs
python3 scripts/build_gp_borough_hubs.py

# Rebuild sitemap
python3 scripts/build_sitemap.py

# Serve locally
python3 -m http.server 8000
```

## SEO

- Every practice has its own static HTML page with full schema.org `MedicalBusiness` + `Physician` + `FAQPage` + `BreadcrumbList`.
- One canonical hub page per borough and per PCN, internally linked.
- Unique meta description on every page including live data ("currently accepting new patients", GPPS score, distance).
- XML sitemap submitted to Google Search Console with weekly `lastmod` updates.
- `robots.txt` with sitemap reference and conservative crawl directives.
